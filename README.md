# TypeRush 🚀

A modern multiplayer typing game built with React, Node.js, Socket.IO, and Zustand.

TypeRush provides multiple typing modes, real-time gameplay, live statistics tracking, and an immersive cyberpunk-inspired UI experience.

---

## 🌐 Live Demo

https://type-rush-frontend-beta.vercel.app/

---

## ✨ Features

- ⚡ Real-time typing experience
- 🎮 Multiple unique game modes
- 📊 Live WPM, Accuracy, Error tracking
- 🔁 Mirror Mode
- 🧠 Memory Mode
- 🎧 Echo Mode
- 🏃 Random Rush
- 🧪 Practice Mode
- 🌙 Cyberpunk UI design
- 🔌 Socket.IO multiplayer support
- 📱 Responsive layout
- ⚙️ Zustand state management

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/AkhilRasamalla/typerushgame-frontend.git
```

---

## Frontend Setup

```bash
cd typerushgame-frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd typenova-backend
npm install
npm start
```

---

## 🔐 Environment Variables

Create `.env` inside frontend:

```env
VITE_BACKEND_URL=http://localhost:3000
```

## 🎮 Game Modes

### Random Rush
Random typing challenges with speed focus.

### Mirror Mode
Words appear reversed while typing.

### Memory Mode
Characters disappear progressively as you type.

### Echo Mode
Typing challenge based on audio memory and rhythm.

### Practice Mode
Focused typing practice with adjustable difficulty.
